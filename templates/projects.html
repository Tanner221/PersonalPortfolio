Projects

Asteroids

This project was a take on the classic atari game "Asteroids". This is a simplified version that is playable. The goal of the user is to destroy all the asteroids and survive without getting hit by an asteroid. This project was built using C++ and graphics from OpenGL


Skeet

This is another game created in C++ and using OpenGL for graphics. The object of this game is to move your rifle (the rectangle in teh bottom corner) and shoot the birds. The birds change color and display numbers based on the amount of health they have. Be careful not to hit the sacred bird!
